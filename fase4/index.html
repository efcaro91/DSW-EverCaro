En esta pagina veremos información pertinente a la tematica de etiquetas basicas de HTML así como sus atributos, desarrollada para el curso de paginas web basico de la universidad nacional abierta y a distancia.

El lenguaje de etiqeutado HTML permite darle formato a el contido de las paginas web, designando mediante cada etiqueta distintos elementos que le permiten al desarrollador determinar la finalidad de cada elemento y el formato que deben tener.
